# Orchestral Machine ("Stanok")

A deterministic, restart-safe, auditable autonomous multi-agent coding system orchestrated with LangGraph.

Cooperating agents take a user-defined task and autonomously architect, implement, review, test, correct, and archive a solution — within strict, bounded controls.

## Status

Working components:
- Core orchestration workflow
- Sandboxed test execution
- Basic operator controls

More detailed documentation and the full codebase are being organized here.
